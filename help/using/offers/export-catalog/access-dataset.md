# Accès au catalogue d&#39;offres exporté {#access-exported-catalog}

Le catalogue d’offres exporté est accessible dans Adobe Experience Platform. **[!UICONTROL Datasets]** . Un jeu de données est créé pour chaque objet de votre bibliothèque d’offres.

![](../assets/datasets-list.png)

Cliquez sur un jeu de données pour accéder à ses détails.

![](../assets/dataset-activity.png)

Le **[!UICONTROL Preview dataset]** vous permet d’afficher le lot le plus récent ayant réussi dans le jeu de données. Le volet de gauche fournit des informations sur les types de données qui ont été exportées.

![](../assets/dataset-preview.png)

Pour plus d’informations sur la navigation et l’utilisation des jeux de données, reportez-vous à la section [cette page](../../data/get-started-datasets.md).
