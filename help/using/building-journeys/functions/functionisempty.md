# isEmpty {#isEmpty}

Renvoie « true » si la chaîne du paramètre ne contient aucun caractère.

## Catégorie

Chaîne

## Syntaxe de la fonction

`isEmpty(<parameters>)`

## Paramètres

* chaîne

## Signature et type renvoyé

`isEmpty(<string>)`

Renvoie une valeur booléenne.

## Exemple

`isEmpty("")`

Renvoie true.

`isEmpty("Hello World")`

Renvoie false.
