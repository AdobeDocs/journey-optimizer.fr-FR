# round {#round}

Renvoie l’entier le plus proche de l’argument avec des arrondis tendant vers l’infini positif.

## Catégorie

Mathématique

## Syntaxe de la fonction

`round(<parameters>)`

## Paramètres

* décimal
* Entier

## Signatures et type renvoyé

`round(<decimal>)`

`round(<integer>)`

Renvoie un nombre entier.

## Exemple

`round(3.14)`

Renvoie 3.

`round(3.54)`

Renvoie 4.

`round(-3.14)`

Renvoie -3.

`round(3)`

Renvoie 3.
