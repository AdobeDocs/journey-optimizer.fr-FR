# upper {#upper}

Renvoie une version en majuscules du paramètre.

## Catégorie

Chaîne

## Syntaxe de la fonction

`upper(<parameters>)`

## Signature et type renvoyé

`upper(<string>)`

Renvoie une chaîne.

## Exemple

`upper("b")`

Renvoie « B ».
