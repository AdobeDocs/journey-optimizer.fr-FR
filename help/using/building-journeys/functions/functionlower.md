# lower {#lower}

Renvoie une version en minuscules du paramètre.

## Catégorie

Chaîne

## Syntaxe de la fonction

`lower(<parameter>)`

## Paramètre

* string

## Signature et type renvoyé

`lower(<string>)`

Renvoie une chaîne.

## Exemple

`lower("A")`

Renvoie &quot;a&quot;.
