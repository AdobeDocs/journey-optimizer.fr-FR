# Référence d’API{#section-overview}

Le canal d’expérience basé sur du code d’Adobe Journey Optimizer vous permet de proposer des offres personnalisées sur vos points de contact numériques avec flexibilité et précision. Que vous utilisiez des outils côté client tels que les SDK web ou mobile d’AEP, les intégrations côté serveur via l’API Edge Network d’AEP ou une approche hybride, cette fonctionnalité s’adapte à vos besoins. Chaque méthode d’implémentation offre des avantages uniques, allant des interactions en temps réel au traitement back-end évolutif. Dans les sections qui suivent, vous allez découvrir les principes de base des Décisions pour les expériences, apprendre à choisir et à mettre en œuvre la méthode appropriée à votre cas d’utilisation et accéder à des ressources pour rationaliser votre parcours. Ouvrons de nouvelles possibilités pour des expériences client personnalisées.
